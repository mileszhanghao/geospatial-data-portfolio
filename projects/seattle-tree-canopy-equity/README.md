# Seattle Tree Canopy Equity Screener

This project analyzes where Seattle experienced tree canopy decline between 2016 and 2021 and whether those losses overlap with environmental justice priority areas. It is framed as an environmental equity screening product for urban planning.

## Methods

- Used Seattle Tree Canopy 2016-2021 census-tract data from Seattle Open Data.
- Mapped relative canopy change by census tract.
- Overlaid environmental justice priority areas to compare canopy loss with equity-relevant geographies.
- Highlighted the ten census tracts with the greatest relative canopy decline.
- Paired the map with a ranked bar chart to make spatial and numerical patterns easier to compare.

## Result

The analysis frames urban tree canopy as civic infrastructure tied to heat, air quality, stormwater, and neighborhood livability. The final report emphasizes how canopy decline is spatially uneven and why the pattern matters for environmental equity planning.
